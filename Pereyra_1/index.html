  • Instrucciones de uso

  • Crear figuras:

    Para crear una figura, seleccioná alguno de los botones ubicados al lado izquierdo del canvas.

    Podés crear círculos, cuadrados, triángulos y rectángulos. Cada vez que presionás un botón, se genera una nueva figura.

    También podés crear figuras de forma aleatoria presionando el botón "Aleatorio".

  • Limpieza del canvas:

    Para borrar las figuras ya dibujadas, simplemente hay que presionar el boton "Borrar" y quedará listo para utilizar.

  • Selección y desplazamiento:

    Uso del mouse

    Para seleccionar una figura, colocá el cursor sobre ella y hacé click izquierdo.

    Para moverla, mantené presionado el click y arrastrá el mouse.

    Uso del teclado

    Primero debés seleccionar la figura con el mouse.

    Luego, mantené presionada la tecla Shift y utilizá las flechas del teclado (arriba, abajo, izquierda y derecha) para moverla.
